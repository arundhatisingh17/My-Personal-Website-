Projects

Welcome to my projects page. Here is a summary of some of the projects I have worked on:

  • WineTasters (HTML, CSS, JavaScript, React, Git, Python, Pandas, NumPy)
    • Implemented an innovative wine reviewing platform featuring interactive sliders for the 5 S’s (See, Swirl, Smell, Sip, Savor), empowering users to intuitively rate wines based on aroma, flavor, and texture.
    • Orchestrated the development of a dynamic user interface displaying aggregated ratings and reviews for popular wines, fostering community engagement and informed purchasing decisions.
    • Spearheaded user testing and feedback iterations to optimize the platform’s usability and functionality, resulting in enhanced user satisfaction and retention rates.
  • MindWI: Mental Health Analysis (Python, Pandas, GeoPandas, Folium, Git, HTML)
    • Spearheaded MindWI, cleaning, sorting, and integrating diverse mental health datasets from Wisconsin and Milwaukee.
    • Employed advanced statistical analysis techniques to uncover trends and correlations, utilizing tools such as Pandas and Matplotlib for insightful data visualization.
    • Leveraged spatial analysis tools including Folium and GeoPandas to create informative maps, empowering stakeholders with actionable insights into mental health trends.
  • CulinaryCompass (Python, mySQL, Web Scraping, DBMS, ReactJS, Flask, Beautiful Soup)
    • Developed and deployed a web application powered by Flask to allow users to explore and discover new recipes based on their preferences and dietary restrictions.
    • Integrated a mySQL database to store a diverse collection of recipes, including ingredients, cooking instructions, and user ratings.
    • Implemented web scraping techniques using Beautiful Soup to continuously update the recipe database with fresh content from popular cooking websites and blogs.
    • Utilized ReactJS to create an intuitive user interface with interactive features such as filtering recipes by cuisine, meal type, or ingredient, providing users with personalized recommendations.
  • Teeko Game Implementation (Python, Pandas, NumPy, Alpha-Beta pruning)
    • Developed a Teeko game implementation in Python, incorporating the Minimax algorithm for decision-making and heuristic evaluation functions for move selection.
    • Implemented an interactive user interface allowing players to make moves and receive feedback on game state and outcomes.
    • Fine-tuned heuristics and optimized the Minimax algorithm for improved performance and strategic gameplay.
